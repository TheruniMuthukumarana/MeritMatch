## Overview

**MeritMatch** is a modern web application designed to connect students, supervisors, and administrators for research project management and collaboration. Built with ASP.NET Core 8, it offers a seamless experience for managing research areas, projects, and user roles.

---

## ✨ Features

- Student, Supervisor, and Admin roles
- Research area and project management
- Secure authentication & authorization
- Responsive UI with Bootstrap
- Dashboard and analytics
- Easy project assignment and tracking

---

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Hiran-Abhisheka/MeritMatch.git
   ```
2. **Open in Visual Studio** and restore NuGet packages.
3. **Update database:**
   ```bash
   dotnet ef database update
   ```
4. **Run the app:**
   ```bash
   dotnet run --project MeritMatch/MeritMatch.csproj
   ```
